Claude Code CLI

Настройка Claude Code CLI

Для CLI можно выполнить готовую команду в PowerShell или вручную создать папку .claude, файл settings.json, env-файл и переменные окружения.

A
Автоматический скрипт

Автоустановка CLI

Откройте PowerShell, замените ключ на свой и выполните команду. Этот вариант настраивает CLI без ручного создания файлов.

Запустить автоустановку 
$h=@{Authorization='Bearer ВАШ_API_КЛЮЧ'}; iex(irm -Headers $h 'https://ru.cheapvibecode.ru/cw')
Замените ВАШ_API_КЛЮЧ на выданный вам API-ключ перед запуском автоустановки. Если вы ввели ключ в окне при открытии сайта, он уже подставлен автоматически.
Перед запуском можно посмотреть содержимое скрипта автоустановки. Команда ниже выводит текст скрипта в PowerShell.
Посмотреть скрипт без запуска 
irm -Headers $h 'https://ru.cheapvibecode.ru/cw'

Ручная настройка CLI

Используйте эти шаги, если хотите самостоятельно проверить и заполнить все файлы.

1

Создайте папку .claude

В профиле пользователя создайте папку для настроек Claude.

Путь 
%USERPROFILE%\.claude

Если папка уже есть, создавать ее повторно не нужно.

2

Создайте settings.json

В папке %USERPROFILE%\.claude создайте файл settings.json.

settings.json 
{
  "env": {
    "ANTHROPIC_BASE_URL": "https://ru.cheapvibecode.ru",
    "ANTHROPIC_AUTH_TOKEN": "ВАШ_API_КЛЮЧ",
    "ANTHROPIC_MODEL": "claude-sonnet-4-6"
  }
}
Замените ВАШ_API_КЛЮЧ на выданный вам API-ключ и сохраните файл. Если вы ввели ключ в окне при открытии сайта, он уже подставлен автоматически.
3

Создайте codex-sale.env

В той же папке создайте файл codex-sale.env и вставьте содержимое ниже.

%USERPROFILE%\.claude\codex-sale.env 
ANTHROPIC_BASE_URL=https://ru.cheapvibecode.ru
ANTHROPIC_AUTH_TOKEN=ВАШ_API_КЛЮЧ
ANTHROPIC_MODEL=claude-sonnet-4-6
В строке ANTHROPIC_AUTH_TOKEN должен быть ваш выданный API-ключ вместо ВАШ_API_КЛЮЧ. Если вы ввели ключ в окне при открытии сайта, он уже подставлен автоматически.
4

Добавьте переменные окружения

Откройте PowerShell и выполните команды ниже, заменив ключ на свой.

В этих командах особенно важно заменить ВАШ_API_КЛЮЧ на ваш выданный API-ключ. Если вы ввели ключ в окне при открытии сайта, он уже подставлен автоматически.
PowerShell 
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL","https://ru.cheapvibecode.ru","User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN","ВАШ_API_КЛЮЧ","User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_MODEL","claude-sonnet-4-6","User")
[Environment]::SetEnvironmentVariable("CODEX_SALE_API_KEY","ВАШ_API_КЛЮЧ","User")

После этого закройте и заново откройте терминал, чтобы он увидел новые переменные.

M

ID моделей

Чтобы переключить используемую модель в CLI, укажите нужный ID в конфиге.

model-id 
claude-fable-5
claude-opus-4-8
claude-opus-4-7
claude-opus-4-6
claude-sonnet-5
claude-sonnet-4-6
claude-haiku-4-5
Для переключения модели замените значение ANTHROPIC_MODEL в settings.json, codex-sale.env или переменных окружения на нужный ID из списка.

Claude App

Настройка Claude App

Для Claude App используйте автоматический PowerShell-скрипт: он сам подготовит нужные файлы и подключит выданный API-ключ.

A
Автоматический скрипт

Автоустановка Claude App

Откройте PowerShell, подставьте свой API-ключ и выполните команду. Этот вариант выполняет настройку приложения автоматически.

Запустить автоустановку 
$env:CODEX_SALE_API_KEY='ВАШ_API_КЛЮЧ'; iex(irm 'https://ru.cheapvibecode.ru/iclaudew')
Замените ВАШ_API_КЛЮЧ на выданный вам API-ключ перед запуском автоустановки. Если вы ввели ключ в окне при открытии сайта, он уже подставлен автоматически.
Перед запуском можно посмотреть содержимое скрипта автоустановки. Команда ниже выводит текст скрипта в PowerShell.
Посмотреть скрипт без запуска 
irm 'https://ru.cheapvibecode.ru/iclaudew'

Баланс

Проверка баланса API-ключа

Вставьте выданный API-ключ, чтобы узнать текущий баланс токенов. Запрос отправляется только на backend этого сайта, а ключ не сохраняется и не записывается в логи.

Проверка

Что проверить после настройки

После сохранения файлов и добавления переменных запустите Claude App или Claude Code CLI заново. Если ключ вставлен корректно, сервис начнет работать с выбранной моделью.

Проверьте ключ

Самая частая ошибка - лишний пробел, неполная вставка ключа или оставленный текст ВАШ_API_КЛЮЧ.

Сверьте имена файлов

Для CLI используется codex-sale.env. Проверьте, что имя файла и переменная ANTHROPIC_MODEL указаны без опечаток.

Перезапустите среду

После настройки CLI заново откройте PowerShell, а после настройки Claude App полностью перезапустите приложение и фоновые процессы Claude.
